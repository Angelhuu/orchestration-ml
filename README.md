# Orchestration ML : Prédiction de résiliation de contrat 

L’objectif est de prédire si un client d’un opérateur télécom va résilier son contrat. La classe 1 correspond aux clients qui quittent l’entreprise (Churn = Yes) et la classe 0 aux clients qui restent (Churn = No). Cette prédiction est utile pour identifier les clients à risque et mettre en place des actions de rétention.
